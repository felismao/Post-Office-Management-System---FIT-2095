## Installation

1. Clone the repository
2. Modify the app.js to connect it with your mongodb database
```shell
const url = 'mongodb://localhost:27017/postoffice';
```
4. Run npm install
```shell
npm install
```
5. Run app.js
```shell
node app.js
```
5. Open localhost:8080
